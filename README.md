# Dsa in C++

A curated collection of C++ implementations for classic Data Structures and Algorithms (DSA).

## Repository Summary

| Section             | Description                                                                                                  |
|---------------------|-------------------------------------------------------------------------------------------------------------|
| **About**           | C++ codes for key data structures and algorithms, ideal for learning, interviews, or competitive programming. |
| **Languages**       | C++                                                                                                          |
| **Topics**          | Arrays, Linked Lists, Stacks, Queues, Trees, Graphs, Sorting, Searching                                      |
| **Features**        | Well-commented code, standalone implementation files, easy compilation instructions                          |
| **License**         | MIT                                                                                                          |

## Core Implementations

| File Name          | Data Structure / Algorithm                | Key Concepts Implemented                |
|--------------------|------------------------------------------|-----------------------------------------|
| array.cpp          | Arrays                                   | Basic array operations, traversals      |
| linkedlist.cpp     | Linked List                              | Singly/Doubly Linked List, insert, delete |
| stack.cpp          | Stack                                    | Push, pop, display, using arrays/LL     |
| queue.cpp          | Queue                                    | Enqueue, dequeue, circular queues       |
| tree.cpp           | Binary Tree, BST                         | Insert, traversals, search              |
| graph.cpp          | Graph                                    | Adjacency list/matrix, BFS, DFS         |
| sort.cpp           | Sorting                                  | Bubble, Selection, Insertion, QuickSort |
| search.cpp         | Searching                                | Linear, Binary Search                   |

## How to Use

| Action        | Instructions                                   |
|---------------|------------------------------------------------|
| **Clone**     | `git clone https://github.com/Veer2401/Dsa-in-Cpp.git` |
| **Compile**   | `g++ filename.cpp -o filename`                 |
| **Run**       | `./filename`                                   |
